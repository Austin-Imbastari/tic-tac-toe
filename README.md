# tic-tac-toe

A small prject utilizing Javascript, CSS, and HTML to create a tic-tac-toe game.

## Technologies Used

- Javascript
- HTML5
- CSS3

## Features

- Users can play against eacother.
- Users can switch to dark mode theme.
- Users can view the winner.
